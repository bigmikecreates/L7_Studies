# APPLICATION LAYER (LAYER 7) STUDIES


This repository focuses on studying first-class concepts of the Application layer (Layer 7) of the OSI Networking Model.
</div>

There are three different entities that live within Layer 7:

1. Application protocols: These define the languages applications use for communication.
2. Application infrastructure: The components of the software that communicate with one another via application protocols.
3. The applications themselves: The aspect of the software that implements business logic instead of infrastructure.
